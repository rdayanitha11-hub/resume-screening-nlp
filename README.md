# 📄 Resume Screening using NLP

AI-powered system that automatically screens resumes and ranks candidates based on job descriptions using Natural Language Processing.

## 🚀 Demo
Live App: Coming Soon

## ✨ Features
- **Upload Resumes**: Supports PDF and DOCX formats
- **NLP Processing**: Extracts skills, education, experience using NLTK/spaCy
- **Match Score**: Calculates similarity between resume and job description
- **Ranking**: Sorts candidates by relevance score
- **Web Interface**: Simple Streamlit app for easy use

## 🛠️ Tech Stack
- **Language**: Python
- **NLP**: NLTK, scikit-learn
- **Web Framework**: Streamlit
- **PDF Parser**: PyPDF2
